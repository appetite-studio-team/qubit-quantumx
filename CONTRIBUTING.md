# Contributing to Qubit Database

Thank you for your interest in contributing. This project is maintained by [Quantumx Foundation](https://quantumx.technology/).

## How to contribute

- **Bug reports and feature requests** — Open a [GitHub Issue](https://github.com/quantumx-foundation/qubit.quantumx/issues). Describe the bug or feature and how to reproduce (for bugs).
- **Code or content changes** — Open a pull request. Keep changes focused and include a short description.

## Getting set up

1. Fork the repo and clone it locally.
2. Run `npm install`.
3. Run `npm run dev` and make your changes.
4. Run `npm run lint` before submitting.
